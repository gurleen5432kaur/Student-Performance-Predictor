# 🎓 Student Performance Predictor

A Machine Learning project that predicts a student's final grade using Linear Regression and Streamlit.

## 🚀 Live Demo

👉 https://student-performance-predictor-g7wfuneq7hg2j7dnlno6vd.streamlit.app/

## 📌 Features

- Predict student performance instantly
- Interactive Streamlit web application
- Linear Regression model
- Easy-to-use interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
